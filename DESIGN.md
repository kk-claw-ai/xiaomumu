# DESIGN.md

## 气质与意象

清晨书房，白纸摊开在橡木桌上，墨绿色钢笔搁在纸边。哑光纸面的微妙颗粒感，空气中只有笔尖触纸的声音。

## 视觉策略

- 白底为主，微妙纸张纹理（CSS 细腻噪点）
- Forest Ink 深绿为唯一 accent，克制使用
- 极少装饰，线条与留白说话
- 无圆角卡片堆叠，不用阴影，不用渐变

## 配色方案

| Token | 值 | 意象 |
|-------|-----|------|
| 背景 | `#FAFAF8` | 哑光纸面微暖白 |
| 前景 | `#1A1A1A` | 炭笔落纸近黑 |
| 辅助 | `#7A7A72` | 纸上铅笔淡痕 |
| Accent | `#1B4332` | Forest Ink 深绿墨水 |
| Accent Hover | `#2D6A4F` | 稍亮的墨绿 |
| 分割线 | `#E2E0DB` | 纸缘折痕 |

## 字体排版

- 标题：Noto Serif SC（宋体），字重 600
- 正文：Noto Sans SC / system-ui，字重 400
- 标题字号克制（h1 = 1.5rem），正文 1rem / 1.75 行高
- 中文优先，英文用 Georgia 衬线呼应标题气质

## 动效与交互

- 板块入口 hover：微下移 2px + 底线渐显
- 页面切换：淡入 300ms
- 列表项 hover：左侧竖线 + 背景微暖

## 设计禁忌

- 不用 emoji
- 不用感叹号结尾
- 不用圆角卡片堆叠
- 不用渐变背景
- 不用阴影（极淡纸纹除外）
- 不用纯黑 #000 / 纯白 #FFF
